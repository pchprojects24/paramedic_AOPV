Placeholder Where and how to record QC results.

Tip: Allow refrigerated test kits to reach room temperature before use (typically 15-30 minutes).
  1. Placeholder Verify test kit is within expiry date and packaging is intact.
  2. Placeholder Don appropriate PPE (gloves at minimum).
  3. Placeholder Collect urine specimen in clean, dry container (first morning void preferred).
  4. Placeholder Remove test device from sealed pouch.
  5. Placeholder Using dropper, apply specified number of drops to specimen well.
  6. Placeholder Start timer immediately.
  7. Placeholder Read result at specified time (do not read after maximum time).
  8. Placeholder Verify control line is present (test is valid only if control appears).
  9. Placeholder Record result immediately.
  10. Placeholder Dispose of test device and specimen in biohazard waste.
