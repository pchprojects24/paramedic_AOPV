Daily Check Summary

Pre-Shift Inspection

PLACEHOLDER

Visual Inspection

  1. Inspect the analyser casing for cracks, moisture ingress, or visible damage.
  2. Check the cartridge port — ensure it is clean, free of dried blood or debris.
  3. Verify the display powers on and shows the main menu without error icons.
  4. Check the IR window is clean (for printer/downloader communication).

Battery Status

  1. Power on the analyser and note the battery indicator.
  2. If the battery icon shows low, replace the 9 V lithium battery.
  3. After replacement, confirm the analyser powers on and retains date/time settings.
Tip: Keep at least two spare 9 V lithium batteries in the medical centre. Alkaline batteries are not recommended — they have shorter life and may leak.

Environmental Check

  1. Verify the medical centre ambient temperature is within 18–30 °C.
  2. Note the current temperature and humidity (if a monitor is available).
  3. If outside range, take corrective action before running tests.

Electronic Simulator Check (EQC)

PLACEHOLDER

Purpose of the EQC

The Electronic Quality Control simulator verifies the analyser's electronic measuring circuits, signal processing, and detection systems without requiring liquid reagents.

Procedure

  1. From the main menu, select the EQC / Simulator option (or insert the EQC cartridge as prompted).
  2. Insert the Electronic Simulator into the cartridge port until it clicks.
  3. Wait for the analyser to complete the electronic test cycle (~60 seconds).
  4. Read the result — the display should show PASS.
  5. Remove the simulator and return it to its protective case.

Interpreting Results

ResultMeaningAction
PASSAnalyser electronics functioning within specificationRecord in logbook, proceed with testing.
FAILOne or more electronic channels out of specificationDo NOT run patient tests. Repeat EQC. If still FAIL, see Troubleshooting.
Caution: Do not run patient samples if the EQC fails. The analyser must pass EQC before any clinical testing is performed.

Reviewing Previous Results

PLACEHOLDER

Accessing Stored Results

  1. From the main menu, navigate to the stored results / data review screen.
  2. Scroll through results from the previous 24 hours.
  3. Note any results flagged with error codes or out-of-range indicators.

What to Look For

  • Repeated error codes on the same cartridge type — may indicate a lot issue.
  • Results that were not printed or transmitted — ensure they are captured.
  • Any results with critical values that may not have been reported.

Cartridge Inventory Check

PLACEHOLDER

Inventory Verification

  1. Check refrigerator stock of each cartridge type — count and record quantities.
  2. Check room-temperature working stock (if cartridges have been removed to equilibrate).
  3. Verify expiry dates — remove any expired cartridges and dispose appropriately.
  4. Confirm lot numbers match the QC logbook records.

Cartridge Equilibration

Important: Refrigerated cartridges must be removed and allowed to reach room temperature (18–30 °C) for at least 5 minutes before use. Do not warm cartridges artificially.

Documentation

PLACEHOLDER

QC Logbook Entry

Record the following for each daily check:

  • Date and time of check
  • Operator name / ID
  • EQC result (PASS / FAIL)
  • EQC simulator lot number
  • Cartridge inventory counts and expiry dates noted
  • Any corrective actions taken
  • Ambient temperature reading (if recorded)
