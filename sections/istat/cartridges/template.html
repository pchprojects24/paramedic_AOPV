QC Workflow

PLACEHOLDER

  1. Verify EQC has passed today.
  2. Check if liquid QC is due for this cartridge type.
  3. If due: run Level 1, Level 2, and Level 3 QC per QC procedures.
  4. Record all results in the QC logbook.
  5. If QC fails, do NOT run patient samples. See QC Failure Troubleshooting.

Patient Sample Workflow

PLACEHOLDER

  1. Confirm patient identity (two identifiers).
  2. Collect the sample: [sample type and collection method].
  3. Open the cartridge foil pouch; remove cartridge.
  4. Apply sample to the cartridge sample well — fill to the indicated line.
  5. Close the cartridge cover.
  6. Insert the cartridge into the analyser.
  7. Enter patient ID and operator ID when prompted.
  8. Wait for the analysis to complete (~[X] minutes).
  9. Read and record the results.
  10. Print the result ticket (if applicable).
  11. Dispose of the used cartridge in the sharps/biohazard container.
  12. Document results in the patient record.
  13. Report critical values immediately per protocol.
