Deep Cleaning Procedure

PLACEHOLDER

Exterior Cleaning

  1. Power off the analyser.
  2. Dampen a lint-free wipe with 70% isopropyl alcohol.
  3. Wipe the entire exterior casing, paying attention to the keypad and display.
  4. Do NOT allow liquid to pool around the cartridge port or IR window.
  5. Allow to air dry completely before powering on.

Cartridge Port Cleaning

  1. Inspect the cartridge port for dried blood, debris, or residue.
  2. Use a dry lint-free swab to gently clean the port contacts.
  3. If contamination is heavy, dampen the swab lightly with isopropyl alcohol.
  4. Ensure the port is fully dry before inserting any cartridge or EQC.

IR Window Cleaning

  1. Locate the IR communication window on the analyser.
  2. Clean with a dry lint-free wipe — do not use solvents on the IR window unless specified by manufacturer.
Caution: Never immerse the analyser in liquid or use abrasive cleaners. Do not autoclave or use ultrasonic cleaning.

Electronic QC Verification

PLACEHOLDER

Procedure

  1. Perform EQC as per the daily EQC procedure.
  2. Confirm PASS result.
  3. If FAIL — clean port, retry. If persistent, refer to Troubleshooting.
  4. Record EQC lot number and result in the weekly section of the QC logbook.
Note: The weekly EQC is identical to the daily EQC but must be recorded separately in the weekly maintenance section of the logbook.

Liquid QC (Level 1, Level 2, Level 3)

PLACEHOLDER

Overview

Liquid QC uses commercially prepared control solutions at known analyte concentrations to verify that the entire analytical process (cartridge + analyser) is performing correctly. Three levels are used to verify performance across the clinical range.

Important: Liquid QC must be run on every cartridge type currently in use. If you carry CHEM8+, CG4+, PT/INR, and Troponin I, you need to run liquid QC on all four types.

Level 1 — Low / Normal

  1. Retrieve Level 1 QC material from refrigerator; allow to equilibrate to room temperature.
  2. Mix the control material per manufacturer instructions.
  3. Select the appropriate cartridge type (e.g., CHEM8+).
  4. Apply the control material to the cartridge sample well.
  5. Insert cartridge into the analyser.
  6. Wait for results; compare to the acceptable range printed on the QC insert.
  7. Record result — PASS if within range, FAIL if outside range.

Level 2 — Mid / Normal

  1. Repeat the above procedure using Level 2 QC material.
  2. Compare results to Level 2 acceptable ranges.
  3. Record in logbook.

Level 3 — High / Abnormal

  1. Repeat the above procedure using Level 3 QC material.
  2. Compare results to Level 3 acceptable ranges.
  3. Record in logbook.

QC Failure Actions

STOP: If liquid QC fails at any level, do NOT run patient samples on that cartridge type until the issue is resolved. See QC Failure Troubleshooting.

Cartridge Lot Verification

PLACEHOLDER

Purpose

Ensure all cartridge lots currently in inventory have been verified with liquid QC and that the correct lot-specific QC ranges are being used.

Procedure

  1. List all cartridge lot numbers currently in stock (refrigerator + room temp).
  2. Cross-reference each lot number against the QC logbook.
  3. Confirm that liquid QC has been run on each lot at all three levels.
  4. If a new lot is found that has not been QC'd, run liquid QC before using for patient testing.
  5. Update the cartridge lot tracking sheet.

Printer Maintenance

PLACEHOLDER

Paper Check & Replacement

  1. Open the printer paper compartment.
  2. Check remaining paper — replace if less than 25% remains.
  3. Install new roll per manufacturer instructions; ensure paper feeds correctly.
  4. Print a test page to confirm.

Print Head Cleaning

  1. Open the printer compartment.
  2. Use a manufacturer-supplied cleaning card or isopropyl alcohol wipe on the print head.
  3. Allow to dry before closing.

IR Communication Test

  1. Align the analyser IR window with the printer IR window.
  2. Initiate a test print from the analyser.
  3. Confirm the printer receives and prints the data.
  4. If communication fails, clean both IR windows and retry. Check alignment distance.

Documentation & Logbook Entries

PLACEHOLDER

Weekly Logbook Entry Requirements

Record the following for each weekly check session:

  • Date and time
  • Operator name / ID
  • EQC result and simulator lot number
  • Liquid QC results for each cartridge type at each level (1, 2, 3)
  • QC material lot numbers and expiry dates
  • Cartridge lot numbers verified
  • Cleaning performed (yes/no, details)
  • Printer status
  • Any corrective actions taken
  • Signature
